# scala-cli-maven-spring-surefire-cucumber-testng-car-data

## Description
A POC for spring app using testng
and cucumber framework with
surefire plugin.

Analyze source code for potential bugs.

## Tech stack
- scala
- maven
  - spring
  - testng
  - surefire
  - cucumber

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
